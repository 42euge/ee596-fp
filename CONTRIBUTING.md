# Contributing

## TunRex Subtree

The `TunRex/` directory is managed as a git subtree from [git@github.com:42euge/TunRex.git](https://github.com/42euge/TunRex).

### Pulling Updates

To pull the latest changes from the TunRex repository:

```bash
git subtree pull --prefix=TunRex git@github.com:42euge/TunRex.git main --squash
```
